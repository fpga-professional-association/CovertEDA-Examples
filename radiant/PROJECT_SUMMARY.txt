All 5 projects build successfully through the full Radiant flow:

Build scripts: .coverteda_build.tcl + .coverteda_build.bat per project

================================================================================
                        CONTENT PER PROJECT
================================================================================

Each project contains:

  src/             Synthesizable Verilog RTL
  constraints/     PDC (pin) + SDC (timing) constraint files
  reports/         Sample vendor report files (for CovertEDA parser testing)
  .coverteda       CovertEDA project configuration (JSON)
  .coverteda_build.tcl   Radiant build script (auto-generated by CovertEDA)
  .coverteda_build.bat   Windows batch wrapper for build execution

================================================================================
                        FIXES APPLIED
================================================================================

FIXED: coeff_rom.v in dsp_fir_filter was not instantiated by fir_top.v.
       Now connected via generate blocks for parallel ROM coefficient access.

FIXED: fir_mac.v was purely combinational (no clock/reset). Rewritten with
       proper registered output for pipelined MAC operations at 50 MHz.

FIXED: UART TX state machine (uart_tx.v) did not reset bit_count or
       sample_count in IDLE state. After the first transmission, bit_count
       stayed at 8, causing subsequent transmissions to skip DATA entirely.

FIXED: UART RX state machine (uart_rx.v) had sample_count transition bugs.
       START checked sample_count == 8 but reset at 7 (unreachable).
       STOP also checked unreachable value. DATA used 8x sampling instead
       of 16x. All fixed to use consistent 16x oversampling with mid-bit
       sampling.

FIXED: SPI flash controller addr input was 32-bit but only 16 bits were
       passed to the FIFO. Reduced addr to 16-bit to match FIFO width.
       Removed unused QSPI and spi_mode ports from spi_top.

FIXED: All SDC files used // comments (invalid TCL syntax). Changed to #.
       Fixed unsupported set_clock_uncertainty dual-arg syntax.

FIXED: All .coverteda config files had wrong backendId ("diamond"),
       wrong topModule ("top_level"), wrong constraint patterns (*.lpf),
       and wrong device ("auto"). All corrected for Radiant/LIFCL-40.
